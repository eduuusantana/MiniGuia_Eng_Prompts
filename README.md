# 📘 Caderno Temático com NotebookLM – Engenharia de Prompts e IA

## 📌 Contexto e Objetivos

Este projeto foi desenvolvido como parte de um desafio prático da DIO, com o objetivo de utilizar a Inteligência Artificial como ferramenta de aprendizagem ativa, explorando o uso do NotebookLM para organização de conhecimento, curadoria de fontes e criação de materiais de estudo.

O tema escolhido para o caderno temático foi **Engenharia de Prompts e uso da IA nos estudos**, por ser uma área essencial para quem deseja trabalhar com tecnologia, programação e ferramentas baseadas em IA.

Objetivos deste projeto:

* Aprender a utilizar IA para auxiliar nos estudos
* Desenvolver pensamento crítico ao analisar respostas da IA
* Praticar engenharia de prompts
* Organizar conhecimento de forma estruturada
* Criar um miniguia reutilizável para revisões futuras
* Utilizar GitHub como portfólio técnico

## 📖 Miniguia de Estudo – Engenharia de Prompts

### ✅ O que é Engenharia de Prompt

Engenharia de prompt é a técnica de criar instruções claras e detalhadas para modelos de Inteligência Artificial, com o objetivo de obter respostas mais precisas, úteis e organizadas.

Ela é muito usada com:

* ChatGPT
* NotebookLM
* Gemini
* Copilot
* Claude
* outras IAs generativas

---

### ✅ Como fazer um bom prompt

Um bom prompt deve ter:

* Contexto
* Objetivo
* Formato desejado
* Nível de explicação
* Exemplo (se necessário)

Exemplo ruim:

Explique IA

Exemplo bom:

Explique Inteligência Artificial em tópicos, com exemplos na programação, para iniciantes
